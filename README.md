# BuyerForeSight User Hub

A premium, fast, and fully responsive User Directory Dashboard built with React, Vite, and completely custom CSS. 

This project implements a futuristic "QuantumSync" theme featuring interlocking hexagon components, dynamic search functionality, and client-side sorting algorithms.

## 🚀 Features
- **Honeycomb Architecture:** Custom CSS layouts generating interactive hexagon user cards that adapt to viewport scaling.
- **Client-Side Data Interrogation:** Instant keyword searching across User Names and Emails.
- **Dynamic Sorting:** Toggle sorting by Name or Company in both Ascending and Descending sequences via custom glowing dropdowns.
- **Full Profile Rendering:** Highly organized nested detailed views utilizing staggered animations for high-end aesthetics.

## 🛠️ Technology Stack
- **Frontend Framework:** React 18 + TypeScript
- **Build Tool:** Vite
- **Routing:** React Router DOM
- **Styling:** Custom Vanilla CSS with Modern Variables (No external UI libraries)
- **Icons:** Lucide React

## 📦 Local Installation & Setup

1. **Clone the repository:**
   ```bash
   git clone https://github.com/kattaRahulkrishna/BuyerForeSight.git
   ```

2. **Navigate into the project directory:**
   ```bash
   cd BuyerForeSight
   ```

3. **Install the dependencies:**
   ```bash
   npm install
   ```

4. **Start the development server:**
   ```bash
   npm run dev
   ```

5. **Build for production:**
   ```bash
   npm run build
   ```

## 🌐 API Source
User data is fetched live from typicode's payload endpoint:
`https://jsonplaceholder.typicode.com/users`
